# LangChain OpenAI Agent

This project demonstrates a simple LangChain-based agent that uses an OpenAI model and custom tools to answer shopping-related questions about product prices and discounts.

## Overview

The agent is built around:

- a LangChain `ChatOpenAI` model
- a small set of custom tools for fetching product prices and applying discounts
- a system prompt that instructs the agent to rely on tool output instead of guessing
- a response extraction helper that formats the final agent result cleanly

## Features

- Single-product price lookup
- Multi-product price comparison
- Discount application for one or more products
- Example execution flow through a hardcoded sample query
- Output written to `output.txt`

## Project Structure

- `main.py` — entry point that creates and runs the agent
- `get_response.py` — utility to extract the final response from LangChain agent output
- `system_prompt.txt` — instructions that guide the agent behavior
- `output.txt` — sample output generated by the script
- `pyproject.toml` — project dependencies

## Requirements

- Python 3.11 or newer
- An OpenAI API key

## Setup

1. Create and activate a virtual environment.
2. Install the dependencies:

   ```bash
   uv sync
   ```

3. Create a `.env` file in the project root and add your API key:

   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```

## Usage

Run the example script:

```bash
uv run main.py
```

This will execute a sample query that compares several products, applies a discount, and writes the result to `output.txt`.

## Notes

The current example focuses on a shopping assistant workflow. You can modify the query inside `main.py` to test different product and discount scenarios.
